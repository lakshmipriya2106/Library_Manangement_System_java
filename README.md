# 📚 Library Management System (Java OOP)

## 📌 Objective

A simple Library Management System built using Java and Object-Oriented Programming concepts.
It allows users to view books, issue books, and return books.

## 🛠️ Technologies Used

* Java
* VS Code
* Terminal

## 🚀 Features

* View all books
* Issue a book
* Return a book
* Menu-driven console application

## 🧠 OOP Concepts Used

* Classes & Objects
* Encapsulation
* ArrayList
* Method interaction between classes

## ▶️ How to Run

```bash
javac LibraryManagementSystem.java
java LibraryManagementSystem
```

## 📂 Project Structure

Single file containing:

* Book class
* User class
* Library class
* Main class
